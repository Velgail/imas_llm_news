シャニソン（アイドルマスター シャイニーカラーズ Song for Prism）の公式運営レター生配信で、今後のコンテンツ展開が明らかになった。

## 6月以降のロードマップ

| 時期 | 内容 |
|------|------|
| 6月〜 | 新SSR Sキャラ取得専用イベント開始 |
| 月次 | 3D MV追加（既存楽曲中心、月1本ペース） |
| 夏 | 夏楽曲・夏コンテンツ展開 |
| 7月末まで | 新SSR Sキャラクター実装継続 |
| 8月以降 | ★3Pアイドルのみ追加（新SSR Sは専用イベントで取得） |
| 11月 | 3周年記念楽曲イベント |

## ガシャ方針変更

月末ガシャには既存楽曲の3D MVと連動したユニット衣装の新★3Pアイドルが登場する方向へシフト。新SSR Sキャラクターは8月以降、通常ガシャでの実装を終了し、専用イベントでの取得形式に一本化される予定。

## REFRAC7IONS チケット締切は今週末

シャニマス（REFRAC7IONS）関連では、「shiny, the first REFRAC7IONS "Still blue"」（12月19・20日 国立代々木競技場 第一体育館）のアソビストアプレミアム会員先行が**今週末6月14日（日）23:59まで**受付中。
