昨日6月6日（土）、横浜アリーナで開催された「Hatsuboshi IDOL FESTIVAL」DAY1にて、初星学園 H.I.F Memorial Single「ガラクタロード」の発売が正式アナウンスされた。

## CD概要

| 項目 | 内容 |
|------|------|
| タイトル | 初星学園 H.I.F Memorial Single「ガラクタロード」 |
| 発売日 | 2026年8月26日（水） |
| 通常盤 | 3,500円（税込） |
| 特装盤 | 5,000円（税込）※アソビストア限定 |

## 収録内容

**通常盤**
- 「ガラクタロード」（13人全員合唱版）
- 「ガラクタロード」ソロ歌唱版 × 13人分
  - 花海咲季、月村手毬、藤田ことね、有村麻央、葛城リーリヤ、倉本千奈、紫雲清夏、篠澤広、姫崎莉波、花海佑芽、十王星南、秦谷美鈴、雨夜燕
- インストゥルメンタル版

**特装盤（ボーナスディスク収録）**
- 「初」「Campus mode!!」「標」の13人全員歌唱フルバージョン

## 予約情報

アソビストアでの予約受付は**6月9日（火）まで**。限定特装盤は在庫に限りがあるため早めの予約が推奨される。

「ガラクタロード」はゲーム内H.I.F編のテーマ曲として5月16日より配信・ダウンロード販売が開始されており、本CDで初めてソロ版が音源化される。
