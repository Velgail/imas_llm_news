アイドルマスター20周年フィナーレを飾る全ブランド越境xRライブ「IDOL WORLD SUPER FESTIVAL 2026（IWSF2026）」に、SideM 315プロダクションから追加出演者が発表された。

## 追加出演者

**アスラン=ベルゼビュートII世**（ユニット：HighxJoker）

DAY2「-ZESSYOU-」（ALLバンド）公演に出演。DAY2は生演奏バンドをテーマにした公演で、各ブランドの選抜アイドルがライブを展開する。

| 項目 | 内容 |
|------|------|
| 開催日 | 2026年7月24日（金）〜26日（日） |
| 会場 | 京王アリーナTOKYO |
| DAY1（-YAKUDOU-） | ALLダンス 開場17:30 / 開演18:30 |
| DAY2（-ZESSYOU-） | ALL生バンド 開場17:00 / 開演18:00 |
| DAY3（-KAGAYAKI-） | コール＆レスポンス／20thフィナーレ 開場17:00 / 開演18:00 |
