本日6月5日（金）11:00、学園アイドルマスターのフェス限ガシャ「ガラクタロード」がvol.2（篠澤広）からvol.3（葛城リーリヤ）に切り替わった。

## ガラクタロードvol.3 概要

| 項目 | 内容 |
|------|------|
| 対象アイドル | 葛城リーリヤ |
| 開催期間 | 6月5日（金）11:00〜6月15日（月）10:59 |
| フェス限種別 | H.I.Fガラクタロード専用フェス限SSR |
| ピックアップ倍率 | 通常比1.5倍 |

## ガラクタロードシリーズ全容

| Vol | 対象アイドル | 期間 |
|-----|--------------|------|
| vol.1 | 月村手毬、莉波 | ～5月26日 |
| vol.2 | 篠澤広 | 5月27日〜6月5日 10:59 |
| **vol.3（本日〜）** | **葛城リーリヤ** | **6月5日 11:00〜6月15日 10:59** |

本ガシャは「H.I.F引き放題ガシャ」（継続可能・最低100連〜）と連動している。シーズン限定を含むPアイドルが対象となる設計で、ガラクタロードシリーズ以外のキャラが「スリぬけ」として出現しない仕様。

メモリアルシングル「ガラクタロード」は全13人による合唱版・ソロ版・ボーナスディスクを収録予定で、詳細は続報待ちとなっている。ジュエル増量セールも6月15日まで継続中。
