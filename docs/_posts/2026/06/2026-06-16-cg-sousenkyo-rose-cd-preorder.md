## 「ROSE ～禁じられた愛～」CD概要

| 項目 | 内容 |
|------|------|
| タイトル | THE IDOLM@STER CINDERELLA MASTER NUMBER ONE! シンデレラNo.1 ＆ ROSE ～禁じられた愛～ |
| 発売日 | 2026年7月1日（水） |
| 歌唱 | 桐生つかさ・木村夏樹・白雪千夜 |
| 作詞作曲・編曲 | 吉見佑昌 / ARM（IOSYS） |
| 予約受付期間 | 5月29日（金）〜**6月21日（日）** |

「ROSE ～禁じられた愛～」は15th Anniversary「シンデレラガール総選挙2026」の応援楽曲として制作された第2弾で、第1弾「シンデレラNo.1」とカップリング収録される。

## 総選挙2026 今後のスケジュール

| フェーズ | 時期 |
|---------|------|
| 投票券ミッション | 7月上旬開始予定 |
| WEB投票 | 8月上旬開始予定 |
| 結果発表 | 未定 |

投票は約190名の対象アイドルからWEB（バンダイナムコID登録・ログイン）で実施。上位7名にソロアルバム「CINDERELLA MASTER」制作と各種グッズ化が決定している。
