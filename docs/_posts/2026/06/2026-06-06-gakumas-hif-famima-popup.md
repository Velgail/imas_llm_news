「横アリ近くのファミマこれで流石に熱い」——そんな一言と写真1枚が、昨日6月6日（土）早朝にXへ投稿され、半日で342万インプレッション・RT8,276・いいね4.1万を集めた。

## 何が起きていたか

ファミリーマート新横浜三丁目店（横浜アリーナ至近）のドリンクコーナーに、手書きPOPが並べられた。内容は学マス（学園アイドルマスター）のゲームシステム「プロデュース」に即したパラメータ効果の表記。

| 飲み物 | ゲーム効果 |
|--------|-----------|
| 水（共通） | パラメータ+10 |
| ウーロン茶（共通） | 元気+7 |
| ホットコーヒー（ロジック） | やる気+3 |
| ルイボスティー（ロジック） | 好印象+3 |
| アイスコーヒー（センス） | 集中+3 |
| ブーストエナジー（センス） | 絶好調2ターン・試験でレッスン強化 |
| ジンジャーエール（アノマリー） | 全力値+1（強度変更） |
| ほうじ茶（アノマリー） | 全力値+2（温泉に変更） |

飲み物を「ロジック」「センス」「アノマリー」という学マスのルート属性で分類した上で効果を割り振るという、ゲームシステムへの深い理解に基づいた構成。棚の上段には「祝！6月6日（土）・7日（日） 学園アイドルマスター 横浜アリーナライブ開催 おめでとう！！」の手書き横断幕、さらに「学園アイドルマスター カードフレームウェーハース ¥415」も開封陳列されていた。

## 反応

投稿主によるリプライで店舗が「**ファミリーマート新横浜三丁目店**」と特定され、「会場近くなので是非行ってあげてくださいませ🙏」と呼びかけられた。

リプライ欄には「これよがしに緑茶買って行きたい」「店外でいろはす捨てられてそう（笑）」「学マス以外でもかなり別のイベントでも陳列凝ってるっぽくてすごえ」など絶賛の声が続出。

## なぜ話題になったか

ゲームを知らなくても一目でわかるユーモア、ゲームを知っていると余計に刺さる属性分類の正確さ、そして「コンビニ店員がそこまでやるか」という驚きが三位一体で刺さった。プロデュースルートの違いをドリンクで表現するという着眼点はファンでも思いつきにくく、大量RTの必然性がある一枚だった。

本日千秋楽のDAY2を前に、横浜の地がアイマスで染まっている証左のひとつ。
