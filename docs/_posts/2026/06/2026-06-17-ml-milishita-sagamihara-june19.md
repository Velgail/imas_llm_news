## コラボキャンペーン概要

| 項目 | 内容 |
|------|------|
| 開始日 | **明日6月19日（金）** |
| 終了日 | 7月31日（金） |
| テーマ | 宇宙を身近に感じられるまち さがみはら |
| 連携 | ミリシタ9周年施策の一環 |

ミリオンライブ！と相模原市は以前にもコラボキャンペーンを実施した実績があり、今回は9周年イヤーに合わせた展開となる。詳細は公式サイト・相模原市観光情報を参照のこと。

## 9周年リアル施策まとめ

ミリシタ9周年では複数の地域・施設とのコラボが進行中・予定されている。

| 施策 | 期間 |
|------|------|
| 相模原市コラボ | 6/19〜7/31 |
| 秋葉原っぱフェスティバル | 6/26〜7/12 |
| カラオケの鉄人コラボ | 6/26〜8/16 |
| ゲーマーズ 9周年シャンパン受注 | 6/26〜7/19 |
