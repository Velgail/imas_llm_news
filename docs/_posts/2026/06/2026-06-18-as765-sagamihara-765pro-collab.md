## 概要

| 項目 | 内容 |
|------|------|
| 期間 | **本日6月19日（金）〜7月31日（金）** |
| エリア | 相模原市内（市立博物館・市立図書館・JAXAキャンパスほか） |
| 参加アイドル | 765PRO ALLSTARS 13名（ソル・ルナ・ステラ） |

765PRO ALLSTARSが天体にちなんだユニット「ソル」「ルナ」「ステラ」として活動していることが、JAXA相模原キャンパスを擁する「宇宙のまち」相模原市とのコラボ実現につながった。

## 五大企画

| 企画 | 内容 |
|------|------|
| JAXAコラボパネル展示 | 相模原市立博物館・市立図書館・JAXA宇宙科学探査交流棟に高さ180cmパネル |
| コラボグッズ販売 | クリアファイル1種・トートバッグ1種・アクリルキーホルダー全13種 |
| ノベルティコラボカード | 協力店舗で500円（税込）以上の購入ごとに全13種ランダム配布 |
| SNSキャンペーン | 相模原市シティプロモーション公式Xのリポストで抽選プレゼント |
| 市内周遊企画 | 市内スポットをめぐるスタンプラリー等 |

グッズの販売場所はさがみはらアンテナショップ sagamix・相模原市立博物館・藤野観光案内所ふじのね。
